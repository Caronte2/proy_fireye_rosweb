Una vez completada la compra de tu unidad Fireye, recibirás en tu correo electrónico un código de activación único. Este proceso garantiza la Máxima Seguridad de tu propiedad:

  1. Identificador Único (ID): Se te asignará una ID de Usuario vinculada exclusivamente al número de serie de tu robot.
  2. Contraseña de Seguridad: Recibirás una clave de acceso encriptada para entrar en tu Dashboard personal.
