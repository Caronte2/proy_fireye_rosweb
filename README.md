# Proy-Fireye-RosWeb
6º Proyecto de grado en tecnologías interactivas.
Interfaz web para el control y monitoreo del robot **Fireye-1**, un sistema de patrullaje autónomo diseñado para entornos rurales y fincas privadas.


## 📋 Descripción
Fireye-1 RosWeb es una aplicación web que permite la comunicación en tiempo real con el robot Fireye-1 a través de ROS (Robot Operating System). Proporciona una interfaz visual para supervisar y controlar las rutas de patrullaje autónomo en entornos al aire libre.


## 🛠️ Tecnologías utilizadas
- HTML5
- CSS3
- JavaScript
- Php
